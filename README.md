# Fake News Detection using Machine Learning

## Objective
To classify news articles as Fake or Real using NLP and Machine Learning.

## Tools Used
- Python
- VS Code
- Pandas, NumPy
- Scikit-learn
- NLTK

## Model Used
- TF-IDF Vectorizer
- Logistic Regression

## How to Run
1. Install requirements
2. Run train.py
3. Run predict.py

## Output
The system predicts whether a given news article is Fake or Real.
